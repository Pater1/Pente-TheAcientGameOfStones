Examples

The example below demonstrates parsing a command line such as "Test.exe /verbose /runId=10"
CopyC#
CommandLineDictionary d = CommandLineDictionary.FromArguments(args);

bool verbose = d.ContainsKey("verbose");
int runId = Int32.Parse(d["runId"]);

Examples

You can also explicitly provide key and value identifiers for the cases that use other characters (rather than '/' and '=') as key/value identifiers. The example below demonstrates parsing a command line such as "Test.exe -verbose -runId:10"
CopyC#
CommandLineDictionary d = CommandLineDictionary.FromArguments(args, '-', ':');  

bool verbose = d.ContainsKey("verbose");
int runId = Int32.Parse(d["runId"]);
